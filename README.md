# Desafio Fullstack 

  Para rodar o projeto, você deve:
    1. Clonar o repositório
    2. Instalar as dependências:
        pip install -r requirements.txt
    3. Rodar:
        set FLASK_APP=application.py (Windows)
        export FLASK_APP=application.py (Mac, Linux)
    4. Por fim, executar:
        flask run
